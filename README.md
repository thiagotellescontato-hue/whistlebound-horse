# Whistlebound Horse

Whistlebound Horse is a Minecraft Fabric mod that lets you whistle to call your selected tamed horse back to you.

After taming a horse, you can choose which one is bound to your whistle using a simple in-game menu. Once selected, press the whistle key and your horse will appear near you in a safe place whenever possible.

## Features

| Feature              | Description                                                                                          |
| -------------------- | ---------------------------------------------------------------------------------------------------- |
| Horse Selection Menu | Press J to choose which of your tamed horses should respond to your whistle.                                    |
| Whistle Keybind      | Press H to call your selected horse.                                                             |
| Safe Horse Placement | The mod tries to place the horse near you without spawning it in walls, lava, fire, or unsafe spots. |
| Saved Horse Data     | The selected horse is saved so it can still be called after moving far away.                         |
| Whistle Sounds       | Plays one of multiple whistle sounds when calling your horse.                                        |
| Client and Server    | Designed to work with the mod installed on both client and server.                                   |

## How It Works

1. Tame a horse.
2. Open the horse selection menu with **J** by default.
3. Select the horse you want to bind to your whistle.
4. Click **Done**.
5. Press **H** by default to whistle and call your selected horse.

![Gui](https://cdn.modrinth.com/data/cached_images/2781e570493fa5e64f62b01c7c74f7935f2209a6.png)

If there is no safe place nearby, the horse will not be moved and you will see a warning message.

## Compatibility

* Minecraft: **1.21.1 through 1.21.11 and 26.1 through 26.1.2**
* Mod loader: **Fabric**
* Environment: **Client and Server**
